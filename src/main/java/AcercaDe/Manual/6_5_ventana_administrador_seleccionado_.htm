6.5 Ventana Administrador Seleccionado.

 
Ventana donde el usuario modifica o elimina el registro seleccionado de la ventana Ver Administrador. Todos los campos estan deshabilitados, y solo se habilitan cuando el usuario presiona el boton de Modificar.
 

Campo de edición

El usuario modifica el nombre del administrador.
Arriba

Campo de edición

El usuario modifica el nombre de usuario (login).
Arriba

Cuadro desplegable

El usuario modifica el tipo de administrador.
Arriba

Cuadro desplegable

El usuario modifica el area o carrera al que pertenece el administrador.
Arriba

Campo de contraseña

El usuario modifica la constraseña actual de administrador.
Arriba

Botón Eliminar

Boton que ejecuta la eliminacion de administrador seleccionado.
Arriba

Campo de contraseña

El usuario escribe la nueva contraseña del administrador.
Arriba

Botón Modificar

Ejecuta la modificacion del administrador seleccionado.
Arriba

Campo de contraseña

El usuario escribe nuevamente la nueva contraseña, para verificar si se ha escrito correctamente.
Arriba

Botón Cerrar

Boton que cancela la modificacion o eliminacion, cerrando la ventana actual.
