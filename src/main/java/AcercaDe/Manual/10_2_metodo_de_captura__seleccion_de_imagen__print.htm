10.2 Método de Captura: Selección de Imagen.

 
 

Selector de Archivos

El usuario deberá buscar la imagen almacenada entre las carpetas.

Botón Aceptar

Cuando la imagen ha sido seleccionada, este aparecerá en la etiqueta a modo de previsualización. Entonces, el usuario al presionar éste botón, la ventana actual se cierra y la imagen se mostrará en la ventana de Ver Servicio o Ver Comunidad.

Botón Desde la cámara

Este botón se deshabilita, puesto que no se necesita en para esté método.

Botón Cancelar

Cancela la selección o captura, cerrando la ventana actual.

Etiqueta

Aquí, la imagen se muestra a modo de previsualización antes su aprobación.
