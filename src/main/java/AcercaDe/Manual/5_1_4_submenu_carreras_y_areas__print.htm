5.1.4 Submenú Carreras y Áreas.

Menu que contiene los accesos directos a las ventanas, donde el usuario podrá registrar, consultar elementos, a la vez que, podrá ejecutar búsquedas de forma personalizada.

JMenu : Menú Carreras y Áreas.

JMenuItem : Agregar Carreras o Áreas.

Botón que ejecuta la ventana donde el usuario registra nuevas carreras o áreas.

JMenuItem : Ver Carreras y Áreas.

Botón que ejecuta la ventana donde el usuario consulta los registros y realiza búsquedas.

El manual de ayuda se crea con Dr.Explain
