10.1 Método de Captura: Cámara Web.

 

Etiqueta

 
Arriba

Botón Desde Archivo

Al seleccionar el método de captura por cámara web, éste botón se deshabilita, pues en este caso, no se necesita.
Arriba

Botón Capturar

Cuando el método de captura por cámara web se elíje, el mismo botón cambia su texto. Al hacer clic en él, la cámara toma la fotografía, cerrando la ventana actual y regresando a Comunidad Seleccionada o Servicio Seleccionado.
Arriba

Botón Cancelar

Cancela la captura de la fotografía, cerrando la ventana actual.
