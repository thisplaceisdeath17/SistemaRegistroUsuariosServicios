10. Ventana Método de Captura.

 
Está ventana presenta ante el usuario los dos métodos de obtención de una imagen, ya se, capturarla por medio de una camara web instalada previamente, o de seleccionar una imagen previamente guardada en los archivos.
 
  • La captura de una fotografía es conveniente cuando se requiere registrar a una comunidad, ya que se necesita esta foto para su impresión en las tarjetas RFID
  • La selección de una imagen almacenada funciona para la identificación de un servicio, puesto que este será mostrado en los kioskos electrónicos.
     
    Aunque la configuración anteriomente dicha, podría ser la estándar, el usuario no tiene prohibido la captura de una fotografía para la identificación de un servicio o seleccionar una imagen almacenada para la identificación de una persona.
     
 

Etiqueta

En está etiqueta se muestra a modo de previsualización, la imagen capturada o seleccionada.

Botón Desde Archivo

Al dar clic en él, se ejecuta un selector de archivos, por el cual, el usuario debe navegar hasta encontrar la imagen que desea

Botón Desde la cámara

Al dar clic en él, la camara web debe de encederse. El usuario podrá ver en tiempo real, su imagen en la etiqueta anteriormente mencionada.

Botón Cancelar

Cancela cualquiera de los dos métodos, cerrando la ventana actual.
