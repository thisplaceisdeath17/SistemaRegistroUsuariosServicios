5.1.2 Submenú Administradores.

 
Menú que contiene los accesos directos hacia las ventanas que permiten el registro, consulta y búsqueda de Administradores del sistema.

JMenu : Administradores

 
Arriba

JMenuItem : Agregar Administrador

Permite la ejecución de la ventana donde se insertan nuevos administradores.
Arriba

JMenuItem : Agregar Tipo de Administradores

Permite la ejecución de la ventana donde se insertan nuevos tipo de administradores.
Arriba

JMenuItem : Ver Administradores

Al hacer clic en este botón, se ejecuta la ventana donde el administrador puede ver y buscar el datos que requiera.
