9. Ventanas Servicios.

 
El módulo de servicios, cuenta con dos ventanas que pueden ejecutarse directamente en la barra de menú, que se encuentra en la ventana principal, dentro del submenú de servicios.
 
Contiene las ventana de Agregar Servicios, encargada de posibilitar el registro de nuevos servicios ofrecidos dentro del plantel. Este formulario, permite la apertura de una ventana donde el usuario puede capturar una fotografía o seleccionar una imagen previamente almacenada.
 
La segunda ventana a la que se tiene acceso es la ventana de Ver Servicios, cuya función radica en mostrarle al usuario todos los registros que están almacenados en la base de datos. Esto le permite al usuario seleccionar cualquier registro para modificarlo o eliminarlo, por medio de otra ventana que solo se ejecuta allí.
