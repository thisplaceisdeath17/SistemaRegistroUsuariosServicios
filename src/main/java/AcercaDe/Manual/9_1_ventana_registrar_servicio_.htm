9.1 Ventana Registrar Servicio.

 

Campo de edición

Campo de texto donde el usuario registra el costo que tiene la adquisición del servicio. Este campo solo admite números.
Arriba

Campo de edición

Campo de texto que almacena el nombre del servicio. Solo admite un máximo de 50 caracteres.
Arriba

Campo de Fecha

Campo de tipo JDateChooser de la librería JCalendar. Contiene la fecha actual del sistema.
Arriba

Campo de Fecha

Campo de tipo JDateChooser de la librería JCalendar. El usuario selecciona una fecha para ejecutar la operación que calcula el número de días que existen entre la fecha actual y la fecha final.
Arriba

Campo de edición

Campo de texto que muestra el resultado del calculo de días entre los dos campos de fecha.
Arriba

Etiqueta

La imagen se muestra en esta etiqueta a modo de previsualización antes de su registro definitivo.
Arriba

Botón Agregar

Ejecuta la inserción.
Arriba

Botón Cancelar

Cancela el registro, cerrando la ventana actual.
Arriba

Campo de edición

El usuario debe escribir una descripción breve acerca del servicio. Tiene un máximo de 250 caracteres.
Arriba

Cuadro desplegable

El usuario debe seleccionar a que área pertenece el servicio a registrar.
Arriba

Botón Agregar Imágen

Ejecuta una ventana para la captura de una fotografía o la selección de una imagen previamente almacenada.
