7.2 Ventana Ver Comunidad.

Etiqueta

 

Campo de edición

El usuario escribe el dato que requiera buscar. Si el campo de texto de deja vacio y se presiona el boton de buscar, se ejecutara una consulta completa de todos los registro en la base de datos. Las busquedas hechas atraves del campos de texto son dinamicas, es decir, se ejecutan busquedas mientras el usuario escribe, hasta dar con el registro que requiere.

Etiqueta

Muestra el numero total de registros en la base de datos.

Tabla

El usuario puede ver todos los registros existentes en la base de datos. Si el usuario desea modificar o eliminar un registro de la tabla, deberá dar doble clic en el, justo en la columna que muestra la matricula/clave.

Cuadro desplegable

El usuario deber escoger el parámetro de búsqueda. Si se deja seleccionado Todos, el sistema realiza una consulta de todos los registros de la base de datos.

Botón Buscar

Ejecuta la búsqueda.
