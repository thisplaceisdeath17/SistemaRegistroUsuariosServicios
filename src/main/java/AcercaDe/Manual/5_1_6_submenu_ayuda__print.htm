5.1.6 Submenú Ayuda.

 
Menú que contiene los accesos directos a los archivo que contienen la ayuda necesaria para que el administrador conozca el sistema.

JMenu : Ayuda

JMenuItem : Acerca de.

Botón que ejecuta un archivo de notas, que contiene información como la versión del sistema, cuándo se le realizó el último cambio, qué necesita el sistema para ejecutarse correctamente, información del desarrollador, etc.

JMenuItem : Ver Manual de Usuario.

Botón que permite la apertura del Manual de Usuario.

El manual de ayuda se crea con Dr.Explain
