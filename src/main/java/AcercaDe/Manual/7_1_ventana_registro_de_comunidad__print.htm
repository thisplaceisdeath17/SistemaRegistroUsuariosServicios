7.1 Ventana Registro de Comunidad.

Campo de edición

Cada miembro de la comunidad es identificada por un numero. Este campo de texto, solo acepta números.

Campo de edición

Es escribe el nombre completo del nuevo registro.

Etiqueta

En esta etiqueta, aparecerá la fotografía capturada o imagen seleccionada, a modo de vista previa, y solo se almacenara cuando el usuario confirme la inserción.

Cuadro desplegable

El usuario deberá escoger que tipo de comunidad esta registrando.

Cuadro desplegable

El usuario deberá seleccionar el area o la carrera al que pertenece el que se registra.

Campo de edición

Este campo almacena el numero de identificación que contiene la tarjeta RFID que acaba de adquirir. Debe ser registrado para ser utilizado como identificador en los kioskos electrónicos.

Botón Agregar

Confirma el registro.

Botón Agregar Fotografía

Ejecuta la ventana para la captura de una fotografía o la selección de una imagen previamente guardada en la máquina.

Botón Cancelar

Cancela el registro, cerrando la ventana actual.
