6.4 Ver Tipos de Administradores.

Etiqueta

Registros almacenado en la base de datos.

Campo de edición

El usuario ejecuta busquedas, escribiendo el dato requerdo.

Tabla

Muestra todos los registro almacenado en la base de datos. Si el usuario desea modificar o eliminar un registro, debe hacer doble clic sobre el registro que prefiera, especificamente en la columna del ID.

Botón Buscar

Ejecuta la busqueda.

Etiqueta

Muestra el numero total de registros almacenados en la base de datos.
