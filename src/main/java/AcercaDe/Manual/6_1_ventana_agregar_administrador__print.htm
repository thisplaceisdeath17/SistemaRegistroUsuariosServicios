6.1 Ventana Agregar Administrador.

Campo de Texto

En este campo de texto, el usuario deberá escribir el nombre de usuario (login) del nuevo registro a insertar.

Combo box

Aquí, todas la áreas y carreras se muestran. El usuario podrá seleccionar el que requiera.

Combo box

El usuario tiene la posibilidad de escoger el rango del nuevo registro.

Campo de Texto (Password)

El usuario que será registrado, deberá escribir su contraseña en este campo. Al ser un campo de texto de tipo Password, lo que escriba, se ocultara.

Botón Agregar

Al hacer clic en él, el registro se habrá concretado.

Campo de Texto

Este campo de texto recibe el nombre completo del nuevo registro.

Campo de Texto (Password)

El nuevo usuario deberá confirma de nuevo su contraseña. El tipo de campo es Password, por lo que, lo que escriba, se ocultará.

Botón Cancelar

Cierra la ventana de registro.
