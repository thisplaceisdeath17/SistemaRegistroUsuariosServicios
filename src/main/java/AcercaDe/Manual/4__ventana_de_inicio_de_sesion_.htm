4. Ventana de Inicio de Sesión.

 
El inicio de sesión verifica que el usuario que desea ingresar, esté registrado en la base de datos. Una cuenta regresiva comienza a correr cuando el usuario a fallado en ingresar sus datos o si el sistema no lo reconoce, al mismo tiempo, tambien aparece las oportunidades se le ofrecen. En caso de que la cuenta regresiva termine, el sistema alertará al posible usuario de que el tiempo se ha agotado y por consecuencia, el sistema se cerrará, obligando al usuario a abrir de nuevo el sistema. Esto mismo sucede cuando sus oportunidades se agotan.
 
En caso de ingresar de forma correcta sus datos, la ventana comenzará a cargar todos los módulos de sistema, y esto se ve reflejado en una barra de progreso.
 
Cuando se termina de cargar, la tercera ventana en mostrarse en la ventana principal.
