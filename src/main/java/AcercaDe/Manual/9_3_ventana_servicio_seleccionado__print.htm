9.3 Ventana Servicio Seleccionado.

 
Esta ventana permite la modificación o eliminación del registro que fue seleccionado en la ventana Ver Servicio. Cuando se ejecuta, todos los campos y botones permanecen inhabiltados hasta que el usuario presiona el botón de modificar. A excepción de los otros controles, el campo de texto del ID de Servicio permance inhabilitado, aun si el usuario dio clic en el botón, ya que este campo ya no se puede modificar.
 

Campo de edición

El usuario no puede modificar el campo del ID.

Campo de edición

El usuario modifica el nombre del servicio.

Campo de Fecha

Campo de tipo JDateChooser de la librería JCalendar, que contiene la fecha actual del sistema, este campo tambien permanece inhabilitado.

Campo de Fecha

Campo de tipo JDateChooser de la librería JCalendar, donde el usuario selecciona una fecha para realizar el calculo de cuanta diferencia existe entre la fecha actual y la fecha seleccionada.

Campo de edición

Campo que muestra el resultado del calculo de días entre los campos de fecha anteriores.

Etiqueta

Muestra la imagen que fue seleccionada para este servicio. Esta etiqueta se actualiza con la nueva imagen si el usuario selecciona una diferente.

Campo de edición

El usuario modifica el precio del servicio.

Botón Eliminar

Ejecuta la eliminación del registro seleccionado.

Botón Modificar

Ejecuta la modificación del servicio seleccionado.

Campo de edición

El usuario modifica la descripción del servicio.

Botón Cancelar

Cancela la eliminación o modificación, cerrando la ventana actual.

Cuadro desplegable

El usuario selecciona el área al que pertenece el servicio seleccionado.

Botón Agregar Imágen

Botón que abre la ventana para la captura de una fotografía o la selección de una imagen previamente guardada en la computadora.
