7. Ventanas Comunidad

 
El módulo de Comunidad, contiene dos ventanas que son accesibles desde la barra de menú en la ventana principal. Estas dos ventanas son:
  • Agregar Comunidad, cuya funcion es la de registrar a la comunidad existente dentro de una empresa o institucion educativa (como es el caso). A traves de esta ventana, el usuario tiene acceso a otra, que tiene como funcion, la captura de una fotografia o la seleccion de una imagen previamente guardada en el PC.
  • Ver Comunidad, es la segunda ventana accesible, que es la encargada de mostrar al usuario todos lo registros almacenados en la base de datos. En esta ventana, el usuario puede acceder a otra, cuya funcion permite la modificacion o eliminacion de dicho registro seleccionado, a la par de, cambiar la fotografia capturada o la imagen seleccionada en la ventana de insercion. En este caso, esta ventana tambien tiene acceso a la interfaz de captura de fotografia y seleccion de imagenes.
