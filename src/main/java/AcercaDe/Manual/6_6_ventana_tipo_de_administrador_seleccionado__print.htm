6.6 Ventana Tipo de Administrador Seleccionado.

 
Ventana donde el usuario puede realizar la operaciones de modificación y eliminación del registro que fue seleccionado en la ventana de Ver Tipos de Administradores. Los campos aparecen inhabilitados la primera vez que se ejecuta está ventana, y solo se habilitan cuando el usuario presiona el botón de modificación.
 

Campo de edición

El usuario no puede modificar este dato. Por lo tanto, el campo de texto permanece inhabilitado.

Botón Eliminar

Ejecuta la eliminación del registro seleccionado.

Campo de edición

El usuario modifica el nombre del nuevo tipo de administrador.

Botón Cancelar

Cancela la modificación o eliminación, cerrando la ventana actual.

Botón Modificar

Ejecuta la modificación.
