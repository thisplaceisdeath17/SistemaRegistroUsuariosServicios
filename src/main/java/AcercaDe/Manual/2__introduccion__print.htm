2. Introducción.

 
SiRiUS TESE es un sistema de administración y gestión de usuarios y servicios, que tiene como objetivo, almacenar a la comunidad de una escuela y de una empresa, así como tambien, el resguardo de los servicios que son ofrecidos a la comunidad.
 
Todos estos datos, son almacenados para una futura consulta en los kioskos electrónicos en donde el personal puede adquirir dicho servicios.
 
Es un sistema que puede adaptarse a cualquier ambiente, es decir, puede ser implementado para una escuela (como es el caso), en una empresa en la que sus empleados obtengan beneficios de algún tipo, etc.
 
Este sistema tiene la capacidad de de seleccionar imagenes para ilustrar mejor los servicios, asi también como la captura de una fotografía para la identificación rápida de la comunidad. Es por ello que, una cámara web deber ser instalada previamente, para su correcto funcionamiento.
 
Este manual de usuario, contiene la descripción de cada uno de las ventanas y de como debe usarse para ejecutar operaciones como inserción, eliminación, consulta y modificación.
