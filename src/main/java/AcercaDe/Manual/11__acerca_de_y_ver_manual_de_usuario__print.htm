11. Acerca de y Ver Manual de Usuario.

 
En la barra de menu, se encuentra un menú llamado Ayuda, que contiene dos opciones a elegir
  • Acerca de, le presenta al usuario un bloc de notas con algunas consideraciones necesarias para la ejecución del sistema, tales como, la compatibilidad de SO., versión del sistema, cual es la fecha de su ultima modificación, información del desarrollador, etc.
 
 
 
2. Ver Manual de Usuario, le muestra al usuario este manual en formato html, para una fácil navegación entre todas las ventanas existentes en el sistema, ayudandolo encontrar lo que necesita saber.

El manual de ayuda se crea con Dr.Explain
