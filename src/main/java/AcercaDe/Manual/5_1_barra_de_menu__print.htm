5.1 Barra de Menú.

 
La barra de menú contiene los accesos a los diferentes módulos que el sistema posée. El usuario podrá dar clic en ellos, para que se muestren el submenú de ventana que podrá seleccionar, para cada uno de los casos.
 
Menú Sistema: Menú que contiene dos botones: Para cerrar la sesión actual y el segundo para cerrar el sistema.
Menu Administradores: Contiene las ventanas para el registro, la consulta y la búsqueda de los administradores.
Menú Comunidad: Contiene las ventanas para el registro, la consulta y la búsqueda de la comunidad.
Menú Carreras y Áreas: Contiene las ventanas para el registro, la consulta y la búsqueda de las carreras y las áreas.
Menú Servicios: Contiene las ventanas para el registro, la consulta y la búsqueda de los servicios.
Menú Ayuda: Contiene dos botones que abren un bloc de notas, para mostrar algunas características del sistema, como la versión, la información personal del desarrollador, etc., y el manual de usuario, en caso de que se necesite.

El manual de ayuda se crea con Dr.Explain
