8.2 Ventana Ver Carreras y Áreas.

 

Campo de edición

El usuario puede buscar el registro que requiera escribiendolo. Este campo de texto ejecuta busquedas dinamicas, es decir, mientras el usuario escribe, se va realizando la búsqueda hasta coincidir con el registro buscado.
Arriba

Cuadro desplegable

El usuario selecciona a traves de que parámetros desea buscar. Si el usuario selecciona todos, se ejecuta una consulta general, llenando la tabla con todos los registros de la base de datos.
Arriba

Tabla

La tabla se llena con los registros de la base de datos. Si el usuario desea modificar o eliminar algún registro, deberá dar doble clic sobre la carrera o área, con la finalidad de que aparezca una ventana donde el usuario aplique estas acciones. El doble clic debe hacerse sobre la columna clave.
Arriba

Etiqueta

Etiqueta

Muestra el número de registros totales en la base de datos.
Arriba

Botón Buscar

Ejecuta la busqueda, dependiendo el filtro que el usuario a seleccionado. Si se ha seleccionado todo, al momento de presionar el botón, se ejecuta la consulta, trayendo con ella, todos lo registro de la base de datos.
