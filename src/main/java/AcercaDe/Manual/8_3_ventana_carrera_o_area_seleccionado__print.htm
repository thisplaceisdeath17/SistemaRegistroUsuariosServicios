8.3 Ventana Carrera o Área Seleccionado.

 
Ventana donde el usuario modifica o elimina el usuario que ha seleccionado de la ventana Ver Carreras y Áreas. Cuando se muestra, todos los campos están inhabilitados. Solo cuando el usuario presiona el botón de modificar, los campos se habilitan, a excepción del campo de la clave, pues este no se puede modificar.
 

Campo de edición

El usuario modifica la clave del área o carrera.

Campo de edición

El usuario modifica el nombre de la área o de la carrera.

Cuadro desplegable

El usuario modifica de que tipo es.

Campo de edición

El usuario modifica el nombre del jefe de carrera o director del área.

Campo de edición

El usuario modifica la descripción del área o la carrera.

Botón Cerrar

Cancela la modificación o eliminación, cerrando la ventana actual.

Botón Modificar

Ejecuta la modificación del registro seleccionado..

Botón Eliminar

Ejecuta la eliminación del registro seleccionado.
