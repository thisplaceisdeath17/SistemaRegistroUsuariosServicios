7.3 Ventana Comunidad Seleccionado.

 
Ventana donde el usuario puede ejecutar la modificación o eliminación del registro seleccionado en la ventana de Ver Comunidad. Todos los campos que pueden editarse esta deshabilitados, y solo pueden ser utilizados cuando el usuario da clic en el botón de modificación. Cabe recalcar que, el campo de la matricula no se habilita, ya que es un campo que no puede modificarse.
 

Campo de edición

El usuario no puede modificar este dato, por lo que el campo permanece deshabilitado.

Campo de edición

El usuario modifica el numero de identificación de la tarjeta RFID, escandeandola por el lector. El lector se encargara de modificar el numero en el chip de la tarjeta.

Campo de edición

El usuario modifica el nombre completo del la comunidad seleccionada.

Etiqueta

Esta etiqueta muestra la imagen o fotografía que fue seleccionada o tomada para este registro, permitiendo asi, una segunda verificación sobre la identidad de a quien se le modifica sus datos.

Campo de edición

El usuario modifica el saldo del la comunidad seleccionada. Este campo solo deber ser modificado, si algún tipo de problema con el crédito ha surgido.

Botón Reporte

El usuario puede generar un reporte de las compras que la comunidad seleccionada ha adquirido. Este reporte solo es generado en caso de algún percance con los servicios o el saldo. A parte de que, la comunidad puede adquirir un reporte a modo de evidencia para algún problema.

Cuadro desplegable

El usuario modifica el area o la carrera al que pertenece la comunidad seleccionada.

Cuadro desplegable

El usuario modifica a que tipo de comunidad pertenece.

Botón Eliminar

Ejecuta la eliminación del registro seleccionado.

Botón Cambiar Fotografía

El usuario modifica la fotografía de la comunidad seleccionada, ejecutando la ventana para la captura de una fotografía o la selección de una imagen previamente almacenada en la máquina. Si no se desea modificar la imagen o fotografía, se debe dejar tal como está.

Botón Modificar

Ejecuta la modificación del registro seleccionado.

Botón Cancelar

Cancela la modificación o eliminación, cerrando la ventana actual.
