5.1.5 Submenú Servicios.

 
Menú que contiene los accesos directos a las ventanas que concede al usuario, la oportunidad de agregar y consultar, a la vez que, el usuario puede ejecutar búsquedas personalizadas.

JMenu : Servicios.

JMenuItem : Agregar Servicios.

Botón que permite la apertura de la ventana donde el usuario inserta nuevos servicios.

JMenuItem : Ver Servicios.

Botón que permite al usuario consultar todos lo registros, ademas de, realizar búsquedas.

El manual de ayuda se crea con Dr.Explain
