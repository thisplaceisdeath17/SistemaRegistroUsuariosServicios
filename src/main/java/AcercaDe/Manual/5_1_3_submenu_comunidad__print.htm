5.1.3 Submenú Comunidad.

 
Menú que contiene los accesos directos a las ventanas que le permiten al usuario insertar, consultar y buscar aquel o aquellos registros que necesita.

JMenu : Comunidad

JMenuItem : Agregar Comunidad

Botón que ejecuta la ventana donde el usuario insertar una nueva comunidad.

JMenuItem : Ver Comunidad

Botón que permite al usuario observar una lista de todos los registro, aparte de, ejecutar búsquedas.

El manual de ayuda se crea con Dr.Explain
