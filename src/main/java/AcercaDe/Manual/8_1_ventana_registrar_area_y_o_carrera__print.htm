8.1 Ventana Registrar Área y/o Carrera.

 

Campo de edición

El usuario otorga un nombre clave al área o a la carrera. Este campo debe ser llenado con letras minúsculas y solo permite 8 caracteres.

Campo de edición

El usuario escribe el nombre de la carrera o el área. El campo de texto solo acepta 60 caracteres.

Cuadro desplegable

El usuario selecciona a que tipo pertenece.

Campo de edición

El usuario registra el nombre completo del jefe de carrera o director del departamento. El campo de texto solo acepta 60 caracteres.

Campo de edición

El usuario escribe una breve descripción del área o la carrera. El área de texto recibe 100 caracteres.

Botón Agregar

Ejecuta el registro.

Botón Cancelar

Cancela el registro, cerrando la ventana actual.
