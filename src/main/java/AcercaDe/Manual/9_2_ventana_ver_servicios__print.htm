9.2 Ventana Ver Servicios.

 

Etiqueta

 

Campo de edición

Campo de texto donde el usuario teclea lo que desea encontrar. Las búsquedas son dinámicas, es decir, mientras el usuario escribe, al mismo tiempo la búsqueda se ejecuta hasta coincidir con el registro deseado.

Botón Buscar

Ejecuta la búsqueda si el campo de texto esta vacío, trayendo todos los registros de la base de datos y llenando la tabla.

Tabla

Los servicios son mostrados en la tabla. Si el usuario desea eliminar o modificar alguno de ellos, debe hacer doble clic en el servicio elegido, concretamente en el área del id. Esto abrirá una nueva ventana donde el usuario podrá llevar a cabo cualquiera de las dos operaciones mencionadas.

Etiqueta

Muestra el número de registros totales en la base de datos.
